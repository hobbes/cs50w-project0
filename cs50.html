There is no better short description for this course than their own words:

Hello, world! This is CS50 (aka CS50x through edX), Harvard University's introduction to the intellectual enterprises of computer science and the art of programming.

This is not a course where you will learn to program and become versatile in any programming language. The focus is more on the way of thinking, the problem solving itself, rather than the technical implementation.

The main teacher for this course is David J. Malan:

There are of course other staffs working and dispensing knowledge, most importantly via short side videos digging deeper into more specific themes. The so called "Shorts". Some of those staffers are:

  • Doug Lloyd
  • Zamyla Chan
  • Brian Yu
  • ...

Almost each lesson has one or more problem sets that the student is expected to be able to solve using the acquired knowledge and other sources. To help with these, there are a number of walkthrough videos, describing the general method to use to reach the solution

Here you will get the basic theoretical knowledge necessary to understand why the things are as they are in CS. The first programs will be written in Scratch, which is a graphical environment making it easy to graps the basics. Then you'll quickly transition to C as the main illustrator for the learned concepts. You'll then get an intro to html and css so you can understand the basics of web development, then transition again to Python, a higher level programming language than C. A bit of SQL and Javascript will add their respective fields of application to your toolbelt.

When you have finished CS50 and are now equipped to understand most of the basics, you can transition to CS50W to get a more in-depth intro to web development.
