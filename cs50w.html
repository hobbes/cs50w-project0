After CS50, you have a basic knowledge of many concepts in the CS field. CS50W proposes to dig deeper in the specific concepts of web development, using the Python and Javascript programming languages

I'm currently taking this course through edX, and am at the very beginning, so I can't yet comment on the specifics, but I can give a short description of the cursus. The main lectures are the following:

  1. Git
  2. CSS
  3. >Flask
  4. >SQL
  5. ORMs, APIs
  6. JavaScript
  7. Front Ends
  8. Django
  9. Testing, CI/CD
  10. GitHub, Travis CI
  11. Scalability
  12. Security

I guess that the staff will be globally similar to the main CS50 course, and that the lectures will be of a similar high quality, but only time will tell :-)
